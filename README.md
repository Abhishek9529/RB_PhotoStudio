# RB Photo Studio Website

A photography studio website built with HTML, CSS and vanilla JavaScript, now set up to run with Vite and npm.

## Pages Included

- Home page
- About page
- Services page
- Gallery page with category filters and lightbox
- Contact page with frontend validation and success message

## Requirements

- Node.js installed on your machine

## Setup

1. Open this project folder in a terminal.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open the local URL shown in the terminal.

## Build

Run:

```bash
npm run build
```

## Preview

Run:

```bash
npm run preview
```

## Deployment

The production build is ready for static hosting on Netlify, Vercel, GitHub Pages, Hostinger, cPanel, or any other static hosting platform.

No backend and no database are required. All paths are relative, and the contact form works on the frontend only.

## Notes

- The website uses dummy studio details and realistic sample names.
- External free stock image links are used for the current visuals.
- The project does not use React, Vue, Angular, Next.js, TypeScript, Tailwind CSS, Express.js, or any frontend framework.
